FAQ

Q: Are my private keys secured?
A: Yes they are secured, your private keys never leave your browser.

Q: How is this secure? Its on a web browser!
A: This wallet has replicated the core features of Zero Classic's daemon in JavaScript! So the process of creating and signing the transactions are all done within the browser. No sensitive information is sent through the network.

Q: Why are you doing this?
A: This wallet was inspired by MyZenWallet & MyBitcoinZ, which was inspired by MyEtherWallet's mission statement: to provide the people the ability to interact with the ZERC blockchain easily, without having to run a full node.

Guide

1. Creating a new wallet

  • Go to https://wallet.zeroclassic.org/
  • Enter a strong but easy to remember secret phrase. Save this somewhere and do NOT forget it.
  • Click the Generate Wallet Button
    • This creates a deterministic wallet based on your secret phrase.

2. Accessing an existing wallet

  • If you've created a wallet on https://wallet.zeroclassic.org/ via the secret phrase, simply re-enter the same phrase.
  • For those with a wallet.zero file generated by GUI wallets. Click settings (), select the Load wallet.zero option and load your wallet.zero file
  • To unlock a single address, Click settings (), select the Paste private key option, paste in your private key and click the unlock button
